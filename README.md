# AI-Enhanced Financial Market Analysis: NVDA vs QQQ

## Overview

This project explores historical market data to analyze the relationship between NVIDIA Corporation (NVDA) and the Invesco QQQ ETF (QQQ). The goal is to apply data analytics techniques to understand price behavior, momentum, volatility, and market trends within the technology sector.

By combining financial market analysis with AI concepts, this project demonstrates how data-driven tools can assist investors and analysts in interpreting market conditions.

## Project Objectives

This analysis aims to answer:

* How has NVDA performed compared to the broader technology sector represented by QQQ?
* What trends and patterns appear in historical price movements?
* How does volatility differ between an individual AI-focused company and the overall tech market?
* Can technical indicators provide insight into momentum and market behavior?
